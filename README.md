I'm currently interested in specializing in data engineering.

Some of the repositories here that you might find interesting:

### Data engineeing

- [**SQL Data Warehouse**](https://github.com/arkadianriver/sql-data-warehouse) - sql-data-warehouse  
  A medallion ETL project resulting from a [_30-hour SQL course by DataWithBaraa_](https://www.youtube.com/watch?v=SSKVgrwhzus).
  I orchestrated the flow with bash and used PostgreSQL instead of SQL Server.
  
### XSLT and DITA-OT

- [**Advent of Code 2024**](ttps://github.com/arkadianriver/AoC-2024) - AoC-2024  
  A start at using XSLT to complete the puzzles in the 2024 Advent of Code challenge, documented with DITA-OT.
  I completed only a few puzzles, but they demonstrate the use of XSLT, with most of the heavy-lifting done with XPath 3.1.
