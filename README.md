## Hi there 👋

My name is Gary and I've been a software developer for 14 years, technical writer for 16.
I'm currently interested in specializing in data engineering.
Some of the repositories I created that you might find interesting:

### Data-focussed

- [**SQL Data Warehouse**](https://github.com/arkadianriver/sql-data-warehouse) - sql-data-warehouse ([project docs](https://arkadianriver.github.io/sql-data-warehouse/))  
  A medallion ETL project resulting from a 30-hour SQL course by DataWithBaraa.
  The SQL queries are mostly his, but I orchestrated the flow with bash and used PostgreSQL instead of SQL Server.

- [**My Money**](https://github.com/arkadianriver/mymoney) - mymoney ([sample reports](https://arkadianriver.github.io/mymoney/))  
  A python module to help keep track of personal financial accounts.
  
### XSLT and DITA-OT

- [**Advent of Code 2024**](https://github.com/arkadianriver/AoC-2024) - AoC-2024 ([project docs](https://arkadianriver.github.io/AoC-2024/))  
  A start at using XSLT to complete the puzzles in the 2024 Advent of Code challenge, documented with DITA-OT.
  I completed only a few puzzles, but they demonstrate the use of XSLT, with most of the heavy-lifting done with XPath 3.1.
